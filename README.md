# Analogue 3D OS Menu

An [ES-DE](https://es-de.org/) theme that translates the look and feel of
**Analogue's 3D OS** into a full multi-system frontend theme.

> Not affiliated with or endorsed by Analogue. "Analogue" and the Analogue OS
> design are the property of their respective owner; this is a fan-made tribute.

## Requirements

- **ES-DE 3.4** or newer

## Features

**Layout variants** (UI Settings → Theme variant)
- Carousel · Carts / Covers / Caps
- List
- Grid · Large / Small
- Gallery ·  Fit / Fill

**Color schemes** (UI Settings → Theme color scheme)
- No Background
- Screenshot, Title Screen, and Fanart (each with Static and Video options)
- Custom — drop your own per-system background into `resources/custom/`
  (named after the ROM folder, e.g. `snesna.png`); see
  [`resources/custom/README.txt`](resources/custom/README.txt)

**Aspect ratios** — 16:9, 16:10, 4:3, 8:7, 1:1, 3:2
(desktop, handhelds, and square displays; 8:7 is tuned for the Retroid Pocket Classic)

**Font sizes** — small, medium, large (UI Settings → Theme font size)

**Languages** — 21 localizations: English, Bosnian, Catalan, German, Spanish,
French, Croatian, Italian, Dutch, Polish, Portuguese (BR/PT), Romanian, Swedish,
Japanese, Korean, Chinese (Simplified/Traditional), Arabic, Russian, Serbian.
